# AgentKey

**Description**: AgentKey is the unified MCP execution layer between AI agents and the real internet. One install gives any AI agent (Claude, Cursor, n8n, Make, or any MCP-compatible client) access to web search, structured scraping, 20+ social platforms (X, TikTok, Reddit, LinkedIn, Xiaohongshu, Instagram, YouTube) plus crypto and on-chain data. One key, one bill, no per-API plumbing.

**Features**:
- Unified MCP server: connect once, access 50+ data tools
- 20+ social platforms (X, TikTok, Reddit, LinkedIn, Xiaohongshu, Instagram, YouTube, Facebook, Discord, Telegram and more)
- Web search & structured scraping (web pages, articles, JSON)
- Crypto market data + on-chain queries
- Works with Claude, Cursor, n8n, Make, ChatGPT, and any MCP client
